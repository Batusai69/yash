Yash

概要

このプログラムは C 言語 (C99: ISO/IEC 9899:1999, JIS X 3010:2003) で書いた Unix 系 OS 用コマンドライン端末シェルです。

これはもともと筆者が学科の課題で作ったプログラムでしたが、 日常の使用に堪えうるものとするために一から再開発中です。

このシェルは POSIX.1-2001 に (ほぼ) 準拠しています。POSIX 準拠を謳っている他のシェル (bash や zsh) よりも精確な POSIX 準拠を目指しています。

なお、yash という名前のシェルは他にも存在かもしれませんが、特に関係はありません。

: このプログラムは SourceForge.jp のプロジェクトの一つとして開発されています。 (→プロジェクト概要ページ)

要件

Software Development Utilities オプションと C-Language Development Utilities オプションを備えた任意の POSIX.1-2001 環境でビルド・実行できるはずです。

開発の進捗状況

すでに POSIX 準拠シェルとして最低限必要な機能は揃っており、一般的なシェルスクリプトを問題なく実行できます。現在、対話モードにおけるコマンドラインの編集・補完機能を実装中です。
